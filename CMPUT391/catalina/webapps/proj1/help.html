Radiology Information System by Alice Wu

The Radiology Information System (RIS) is a web-based database application system, which resembles the real life radiology information system.

    A Radiology Information System (RIS) is a computer information system that stores and processes the information for a radiology lab, including all the data regarding patients, and all medical images generated by various diagnostic and therapeutic equipments, such as X-ray, Computed Tomography (CT/CAT), Ultrasound (US), Magnetic resonance imaging (MRI). A Radiology Information System is optimally complemented with a Picture Archiving and Communication System (PACS).

    In general, the digital images generated by radiology equipments shall be in the format of Digital Imaging and Communications in Medicine (DICOM). (DICOM is the industry standard for transferal of radiologic images and other medical information between computers.) For this project, however, a jpg format will be used instead.

Install Instruction
  1. Make sure you have installed tomcat in your system.

The radiology information system has four types of users:

Administrator

  • Sign up a new user
  • Update user information on an existing user
  • Add a new family doctor relationship
  • Generate a report based on a specified diagnosis and a given time period
  • Search users with keywords and/or a given time period
  • Data analysis module

Doctor

  • Change his/her own personal information
  • Search his/her patients with keywords and/or a given time period

Radiologist

  • Change his/her own personal information
  • Search his/her patients with keywords and/or a given time period
  • Create a new radiology record

Patient

  • Change his/her own personal information
  • Search his/her records with keywords and/or a given time period

Log In

    Please Enter your username in the UserName block.

    Please Enter your password in the Password block.

    After finishing above steps, click on LogIn button

    On success, you will be directed to your homepage. Or else, you'll have to reenter your username and password.


Homepage

    On homepage, you'll be given choices to change your personal information including password, firstname, lastname, address or phone number.

    Also, you could click on "Enter Search Engine", to search radiology records in the system

    To safely log out, you need to click on Log Out button.


Admin Homepage

    You could register a new user by clicking on "Register a New User" button.

    And by entering a valid person id and clicking on the "Update a User" will direct you to a page on which you could change the user's personal information (Of course, you could change your information on this page!)

    "Update Family Doctor Information" will allow you to add a new family doctor relationship or delete a relationship.

    Also, you could click on "Enter Search Engine", to search radiology records in the system

    We also provide a report generating module, in which you could get a list of all patients with a specified diagnosis in a given time period by simply clicking on "Generate a report" button.

    To safely log out, you need to click on Log Out button.


Sign Up

**You must be an administrator to perform this job**

  1. Decide if you are going to register a new account for an existing user or a new user by select one of the options(Existing User & New User) with the radio button
  2. Enter the username of the new account in UserName block. The limit of characters is 24.
  3. Enter the password of the new account in Password block. The limit of characters is 24 and no special character is allowed.
  4. Enter the first name of the new account in First Name block. The limit of characters is 24 and no special character is allowed.
  5. Enter the last name of the new account in First Name block. The limit of characters is 24 and no special character is allowed.
  6. Choose a role for the new account (a patient, radiologist or doctor?)
  7. Enter the email of the new account in Email block.
  8. Enter the address of the new account in Address block. The maximum number of characters is 128.
  9. Enter the phone number of the new account in Phone block. A valid phone number must consist of 10 digits. No special characters, spaces, or dashes.
  10. After filling in all the blanks, click on SignUp button.

Update Personal Information

**This part is for administrators only**

The user's information will be populated into the personal information form. Please note that you need to enter a new password to change his/her password or enter his/her odd password to confirm not changing his/her password.

  1. Enter the new Username in UserName block if you would like to change the user's username.
  2. Enter the new password to change the user's password. Or Enter the odd password in order to keep the odd password.
  3. Enter the new first name of the user's if you would like to change the user's first name.
  4. Enter the new last name of the user's if you would like to change the user's last name.
  5. Select a role for the user by click on radio buttons of patient or radiologist or doctor.
  6. Enter the new email address if you need to change the user's email address.
  7. Enter the new address if you need to change the user's address.
  8. Enter the new phone number if you need to change the user's phone number.
  9. Click Save button to save or Go back button to discard your change.
**This part is for doctors/patients/radiologists**
  • Change Password: click on Change Password button first. Then entering your old password and new password to make this change.
  • Change Firstname: click on Change Firstname button and then enter your new first name in New Firstname block at the bottom of this page. Click save to save your change.
  • Change Lastname: click on Change Lastname button and then enter your new last name in New Lastname block at the bottom of this page. Click save to save your change.
  • Change Address by clicking Change Address button, and enter the new address in the New Address block at the bottom of this page. Click Save to save your change.
  • Change Phone number by clicking on Change Phone Number button. And enter your new phone number in New Phone block at the bottom of this page. Click Save button to save your change.

Update Family Doctor Information

**You must be an administrator to perform this job.**

    When at the page of Edit Family Doctor Information, you are provided with a table of current family doctor relationships.

    You could either add a new relationship or drop an odd one.

  1. Entering a valid doctor's person id in the first block
  2. Entering a valid patient's person id in the second block
  3. Then click on "Add relationship" to add this new relationship to system or "Drop relationship" to delete this relationship from system.

Generate a Report

**You must be an administrator to perform this job.**

  1. Enter a specific diagnosis (eg. lung cancer) in the first blank
  2. Enter a time period in the from and to blank strictly following the given format (eg. 29-FEB-2008)
  3. Then click Go to generate this report.

Search Engine

    Search Engine first lists all the records that are available to users.

    (Due to security concerns, doctors can only search radiology records of his patients', and radiologists only have access to records conducted by themselved, and patients can only see their records.)

    Search Engine will take in a list of keywords joined by only and or or and a time period and/or the order of results.

  1. First enter the keywords in the first blank. (eg. Jully and cancer and Scully, Slim or Shady)
  2. Enter a time period. Enter the start date in From blank, and the end date in the To blank. (Not: please strictly follow the format of the date, and the search engine will do the search including the date entered in the From blank and excluding the date entered in the To blank)
  3. (Optional) Select the order that you would like to see your results. The available orders are "Most recent first" and "Least recent first". If leave this blank, the results will be listed by the default order of our system.
  4. Click Go to perform the search

Data Analysis Module

**You must be an administrator to perform this job.**

    Data Analysis Module is used by a system administrator to generate and display an OLAP report for data analysis. It will display the number of images or records for each patient, test type, and/or period of time.

  • Select whether you would like to see the number of images or records by selecting the radio button.
  • Test type is optional. You could leave the box unchecked if you do not need the information.
  • Specify the time period by check one of "day", "week", "month", "year". This part is also optional.
  • Click "Go" button when finish. The result will be displayed on current page.
