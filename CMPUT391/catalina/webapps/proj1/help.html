Radiology Information System by Alice Wu

The Radiology Information System (RIS) is a web-based database application system, which resembles the real life radiology information system.

    A Radiology Information System (RIS) is a computer information system that stores and processes the information for a radiology lab, including all the data regarding patients, and all medical images generated by various diagnostic and therapeutic equipments, such as X-ray, Computed Tomography (CT/CAT), Ultrasound (US), Magnetic resonance imaging (MRI). A Radiology Information System is optimally complemented with a Picture Archiving and Communication System (PACS).

    In general, the digital images generated by radiology equipments shall be in the format of Digital Imaging and Communications in Medicine (DICOM). (DICOM is the industry standard for transferal of radiologic images and other medical information between computers.) For this project, however, a jpg format will be used instead.

Install Instruction

  1. Make sure you have installed tomcat in your system.
  2. Create/Edit .catalinarc in your home directory. Modify CLASSPATH to "CLASSPATH='[your oracle jdbc library]:~/catalina/webapps/proj1/WEB-INF/lib/*'". Comment out the lines 'JAVA_HOME=/usr/local/java' and 'export JAVA_HOME' by adding a '#' symbol at the beginning of these two lines (if two lines exist). Also, comment out LD_LIBRARY_PATH and export LD_LIBRARY_PATH if applicable.
  3. Extract and move the whole folder "proj1" into webapps folder. And open the terminal, cd to proj1 directory, type in make. Then your installation is done.
  4. Before using the application, please restart your tomcat.

The radiology information system has four types of users:

Administrator

  • Sign up a new user
  • Update user information on an existing user
  • Add a new family doctor relationship
  • Generate a report based on a specified diagnosis and a given time period
  • Search users with keywords and a given time period
  • Data analysis module

Doctor

  • Change his/her own personal information
  • Search his/her patients with keywords and a given time period

Radiologist

  • Change his/her own personal information
  • Search his/her patients with keywords and a given time period
  • Create a new radiology record

Patient

  • Change his/her own personal information
  • Search his/her records with keywords and a given time period

Log In

    Please Enter your username in the UserName block.

    Please Enter your password in the Password block.

    After finishing above steps, click on LogIn button

    On success, you will be directed to your homepage. Or else, you'll have to reenter your username and password.


Homepage

    Homepage is a module that provides doctors, patients, and radiologists to modify his/her personal information or explore the functions provided by Radiology Information System, such as the search engine.

    On homepage, you'll be given choices to change your personal information including password, firstname, lastname, address or phone number.

  • Passwords are consist of numbers and letters. Special characters are illegal.
  • A person's first name and last name should contain only letters.
  • The maximum length of address is 128 characters.
  • A valid phone number should contain no spaces but only 10 digits.

    Also, you could click on "Enter Search Engine", to search radiology records in the system. For more information on search engine, please refer to search engine section.

    For radiologists only, the function "Create a New Radiology Record" is available for radiologists to add a new radiology record to system.

    To safely log out, you need to click on Log Out button.


Admin Homepage

    Admin homepage is a place, in which an administrator will perform jobs such as modifying a user's information, signing up a new user, updating family doctor relationship, etc.

    You could register a new user by clicking on "Register a New User" button.

    And by entering a valid person id and clicking on the "Update a User" will direct you to a page on which you could change the user's personal information (Of course, you could change your information on this page!)

    "Update Family Doctor Information" will allow an administrator to add a new family doctor relationship or delete a relationship.

    Also, an administrator could click on "Enter Search Engine", to search radiology records in the system

    We also provide a report generating module, in which an administrator could get a list of all patients with a specified diagnosis in a given time period by simply clicking on "Generate a report" button.

    To safely log out, you need to click on Log Out button.


Sign Up

**You must be an administrator to perform this job**

    Sign Up module is used by an administrator to register a new user in Radiology Information System. The registration form requires the username, password, class (role), the information that if the user is an existing user or a new user.

  1. Decide if you are going to register a new account for an existing user or a new user by select one of the options(Existing User & New User) with the radio button
  2. Enter the username of the new account in UserName block. The limit of characters is 24, only letters and numbers are allowed.
  3. Enter the password of the new account in Password block. The limit of characters is 24, and should only contain letters and numebrs.
  4. Enter the first name of the new account in First Name block. The limit of characters is 24 and no special character or number is allowed.(Optional)
  5. Enter the last name of the new account in First Name block. The limit of characters is 24 and no special character or number is allowed.(Optional)
  6. Choose a role for the new account (a patient, radiologist or doctor or admin?)
  7. Enter the email of the new account in Email block. (Optional)
  8. Enter the address of the new account in Address block. The maximum number of characters is 128.(Optional)
  9. Enter the phone number of the new account in Phone block. A valid phone number must consist of 10 digits. No special characters, spaces, or dashes.(Optional)
  10. After filling in all the blanks, click on SignUp button.
    • If registering an existing user, a dialog window will prompt you for his/her person id. Please only enter legal person id. Else the window will keep asking for the correct person id.
    • Otherwise a new person id will be assigned to the new user automatically.

Update Family Doctor Information

**You must be an administrator to perform this job.**

    When at the page of Edit Family Doctor Information, administrators are provided with a table of current family doctor relationships for reference. And an administrator is able to add a new family doctor relationship or delete one by entering valid doctor and patient id.

  1. Entering a valid doctor's person id in the first block
  2. Entering a valid patient's person id in the second block
  3. Then click on "Add relationship" to add this new relationship to the system or "Drop relationship" to delete this relationship from the system.

Create a New Radiology Record
  1. First fill in all necessary information that required by a radiology record, eg. patient id, doctor id, diagnosis, description, etc.
  2. By clicking on save button, the system will prompt you for uploading images. If no images to upload, click "No image to upload" button. Or else, select multiple files or a single file to upload.
  3. After uploading, the system will ask you if you would love to upload more. Clicking yes would continue your uploading process. Otherwise, you will be directed to your homepage.

Generate a Report

**You must be an administrator to perform this job.**

    The "Generate a Report" Module is responsible for taking in a specific diagnosis and a time period, returning a list of patients that qualify the diagnosis in their radiology records.

    • Please note: report is generated based on a very specific diagnosis. This is not like in search module where multiple keywords are acceptable.
    • If entering "cancer", then the people that has records that contains word "cancer" in their diagnosis column will be returned.
    • If entering "cancer and cold", then people who has records that contains word "cancer and cold" in the diagnosis column will appear in the result table
  1. Enter a specific diagnosis (eg. lung cancer) in the first blank
  2. Enter a time period in the from and to blank strictly following the given format (eg. 29-FEB-2008)
  3. Then click Go to generate this report.

Search Engine

    Search Engine first lists all the records that are available to users.

    (Due to security concerns, doctors can only search radiology records of his patients', and radiologists only have access to records conducted by themselves, and patients can only see their records.)

    Search Engine will take in a list of keywords joined by only and or or and a time period and/or the order of results.

  • "and" and "or" are reserved keywords, which should never be used as a search word.
  • A time period refers to a time interval with a start date and an end date.
  1. First enter the keywords in the first blank. (eg. Jully and cancer and Scully, Slim or Shady)
  2. Enter a time period. Enter the start date in From blank, and the end date in the To blank. (Not: please strictly follow the format of the date, and the search engine will do the search including the date entered in the From blank and excluding the date entered in the To blank)
  3. (Optional) Select the order that you would like to see your results. The available orders are "Most recent first" and "Least recent first". If leave this blank, the results will be listed by the default order of our system, which is calculated by formula Rank(record_id) = 6*frequency(patient_name) + 3*frequency(diagnosis) + frequency(description), with the largest rank appearing first.
  4. Click Go to perform the search

Data Analysis Module

**You must be an administrator to perform this job.**

    Data Analysis Module is used by a system administrator to generate and display an OLAP report for data analysis. It will display the number of images for each patient, test type, and/or period of time.

  • Test type is optional. You could leave the box unchecked if you do not need the information.
  • How to select a time period?
    • Select day will show the result of number of images based on each day.
    • Select week will show the result of number of images based on each week.
    • Select month will show the result of number of images based on each day
    • Select year will show the result of number of images based on each year
    • Select exact time will show the result of number of images based on an exact time of your choice
      • You can choose a single week, month, year or date.
      • Or combine week and/or month and/or year.
      • When specifying a date, please strictly follow the format 'dd-MON-YYYY'.
    • Select a time period will show the result of number of images based a time period of your choice
      • If only want the result after a week/month/year/date, please using only "From week/month/year/date".
      • If only want the result before a week/month/year/date, please using only "To week/month/year/date".
      • When selecting a time period with both "from" and "to" selections, please make sure the format of your selected date is correct.
        (eg. From week 12 From year 2011 + To week 16 To year 2012, From week 12 + To week 16)
      • Or combine week and/or month and/or year.
      • When specifying a date, please strictly follow the format 'dd-MON-YYYY'.
  • Click "Go" button when finish. The result will be displayed on current page.
